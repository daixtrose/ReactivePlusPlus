Recover from an on\+\_\+error notification by continuing the sequence without error. 

The operator intercepts an {\ttfamily on\+\_\+error} notification from the source Observable and, instead of passing it through to any observers, replaces it with some other item or sequence of items. \begin{DoxyWarning}{Warning}
This operator potentially allows the resulting Observable to terminate normally or not to terminate at all.
\end{DoxyWarning}


Actually this operator just subscribes on observable calculated from callback when {\ttfamily on\+\_\+error} from original observable obtained.
