Observer specific with types of callbacks to avoid extra heap usage. 

It has better performance comparing to \mbox{\hyperlink{classrpp_1_1dynamic__observer}{rpp\+::dynamic\+\_\+observer}}. Use it if possible. But it has worse usability due to On\+Next/\+On\+Error/\+On\+Completed template parameters.
