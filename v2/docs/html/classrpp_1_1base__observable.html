Detailed Description

template<constraint::decayed_type Type, constraint::observable_strategy< Type > Strategy>
class rpp::base_observable< Type, Strategy >

Base class for any observable used in RPP. It handles core callbacks of observable.

Observable provides ony core function: subscribe - it accepts observer (or any way to construct it) and then invokes underlying Strategy to emit emissions somehow.

Warning
Actually observable "doesn't emit nothing", it only invokes Strategy! Strategy COULD emit emissions immediately OR place observer to some place to obtain emissions later (for example subjects)
Template Parameters
Typeof value this observable would provide
Strategyused to provide logic over observable's callbacks
