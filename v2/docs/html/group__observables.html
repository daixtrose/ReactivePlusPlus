Detailed Description

Observable is the source of any Reactive Stream.

Observable provides ability to subscribe observer on stream of events. After subscription observable would emit values to observer.

See also
https://reactivex.io/documentation/observable.html

template<constraint::decayed_type Type>
using rpp::dynamic_observable = typedef base_observable<Type, details::observable::dynamic_strategy<Type> >

Type-erased version of the rpp::base_observable. Any observable can be converted to dynamic_observable via rpp::base_observable::as_dynamic member function.

To provide type-erasure it uses std::shared_ptr. As a result it has worse performance.
