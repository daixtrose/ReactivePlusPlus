Converts observable of observables of items into observable of items via merging emissions. 

\begin{DoxyWarning}{Warning}
According to observable contract (\href{https://reactivex.io/documentation/contract.html}{\texttt{ https\+://reactivex.\+io/documentation/contract.\+html}}) emissions from any observable should be serialized, so, resulting observable uses mutex to satisfy this requirement

During on subscribe operator takes ownership over \doxylink{classrpp_1_1schedulers_1_1current__thread}{rpp\+::schedulers\+::current\+\_\+thread} to allow mixing of underlying emissions
\end{DoxyWarning}


Actually it subscribes on each observable from emissions. Resulting observables completes when ALL observables completes

\begin{DoxyParagraph}{Performance notes\+:}

\begin{DoxyItemize}
\item 2 heap allocation (1 for state, 1 to convert observer to dynamic\+\_\+observer)
\item Acquiring mutex during all observer\textquotesingle{}s calls
