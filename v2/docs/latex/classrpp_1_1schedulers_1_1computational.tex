\doxysubsection{Detailed Description}
Scheduler owning static thread pool of workers and using "{}some"{} thread from this pool on {\ttfamily create\+\_\+worker} call. 

\begin{DoxyWarning}{Warning}
Actually it is static variable to {\ttfamily \doxylink{classrpp_1_1schedulers_1_1thread__pool}{thread\+\_\+pool}} scheduler 
\end{DoxyWarning}
\begin{DoxyNote}{Note}
Expected to pass to this scheduler intensive CPU bound tasks with relatevely small duration of execution (to be sure that no any thread with tasks from some other operators would be blocked on that task)
\end{DoxyNote}
