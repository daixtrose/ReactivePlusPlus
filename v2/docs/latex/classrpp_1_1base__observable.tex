\doxysubsection{Detailed Description}
\subsubsection*{template$<$constraint\+::decayed\+\_\+type Type, constraint\+::observable\+\_\+strategy$<$ Type $>$ Strategy$>$\newline
class rpp\+::base\+\_\+observable$<$ Type, Strategy $>$}
Base class for any observable used in RPP. It handles core callbacks of observable. 

Observable provides ony core function\+: subscribe -\/ it accepts observer (or any way to construct it) and then invokes underlying Strategy to emit emissions somehow. \begin{DoxyWarning}{Warning}
Actually observable \char`\"{}doesn\textquotesingle{}t emit nothing\char`\"{}, it only {\bfseries{invokes Strategy!}} Strategy COULD emit emissions immediately OR place observer to some queue or something like this to obtain emissions later (for example subjects)
\end{DoxyWarning}

{\em Type} & of value this observable would provide. Only observers of same type can be subscribed to this observable. \\
\hline
{\em Strategy} & used to provide logic over observable\textquotesingle{}s callbacks. \\

\end{DoxyTemplParams}
